Que es CSS

CSS es el lenguaje que se usa para dar estilo a una página web, incluyendo el diseño y las variaciones en la pantalla para diferentes dispositivos y tamaños.

En este último ejemplo, podemos observar que es posible agregar código CSS dentro del archivo HTML, a este tipo de estilo se le conoce como “inline CSS”. IMPORTANTE: Inline CSS Es cuando se usa un estilo para aplicar a un solo elemento. Para usar estilos en línea, se agrega el atributo style al elemento HTML. El atributo style puede contener cualquier propiedad CSS. 5. RGBA y 6. HSLA En estos casos los rgba y hsla son extensiones de rgb y hsl respectivamente con la diferencia que al final se le agrega un valor que hace referencia al canal alfa, que especifica la opacidad del color. El valor del canal alfa oscila entre 0 y 1. Ejemplo: Utilizar el color tomato, para agregar colores de fondo a etiquetas h1, además colocar otros dos en formato rgba y hsla con transparencia al 50%.
